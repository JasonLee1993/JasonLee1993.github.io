Adding a border to an image

Result

isn't she beautiful?!

Algorithm

We find the specific position of the pixels that we wanna define as border, set their R,B,G values as black

Code

     var image = new SimpleImage("647318356576520916.jpg");
print(image);
var w = image.getWidth();
var h = image.getHeight();
for(var p of image.values()){
    
    if(p.getX()>(w-11) || p.getX()<10 || p.getY()>(h-11) || p.getY()<10){
    p.setBlue(0);
    p.setGreen(0);
    p.setRed(0);
    }
}

print(image);
